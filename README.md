# Quizzler

Викторина с вопросами True/False. Реализация MVC паттерна.

## Функционал

- 13 вопросов с ответами True/False
- Подсчёт очков (Score)
- Прогресс-бар
- Визуальная обратная связь (зелёный/красный)
- Автоматический переход к следующему вопросу
- Сброс викторины после завершения

## Технологии

- Swift
- UIKit
- MVC (Model-View-Controller)
- Timer
- UIProgressView
- Struct (QuizBrain, Question)

## Структура MVC

- **Model**: Question.swift, QuizBrain.swift
- **View**: Main.storyboard
- **Controller**: ViewController.swift
